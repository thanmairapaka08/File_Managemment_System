# Java File Management System

## Overview
This project is a Java-based File Management System that allows users to perform basic file operations such as creating, writing, reading, and deleting files through a **menu-driven console application**.

The project demonstrates practical usage of Java File I/O and object-oriented programming concepts.

---

## Features
- Create a new file
- Write content to a file
- Read file content
- Delete a file
- Console-based user interaction

---

## Technologies Used
- Java  
- File I/O ('File', 'FileWriter', 'Scanner')  
- Exception Handling  
- Object-Oriented Programming  

---

## How It Works
1. The user selects an operation from the menu.
2. The system performs the selected file operation.
3. Output is displayed in the console.
